# Arduino-Flappy-Dot
Simple Flappy bird style game on an arduino and an 8x8 Dot matrix display!

## Components: 
* Arduino board (any should do)
* Breadboard
* MAX7219 DotMatrix Module
* Button set as pull-up resistor
* 5 F-M Cables
* 7 M-M Cables
